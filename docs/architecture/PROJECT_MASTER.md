## 1. What This Project Is

A private, manually-executed, live-data stock + crypto screening platform (Shariah-compliance aware for stocks). Users configure indicator/filter combinations (RSI, WaveTrend, Aroon, LRC, Regression Channel, Trend Channel, volume/price filters, channel-respect and confluence post-filters), click Run, and get a non-interactive results table ‚Äî no charts, no buy/sell signals, no auto-refresh. Backend is FastAPI (`backend/`), frontend is React+Vite+TypeScript with Firebase auth (`frontend/`). Full architecture and file map: [`Documentation.md`](./Documentation.md).

**Client's own one-line definition (from the spec):** *"A private, manually executed, live-data screening system that evaluates stocks and crypto using user-defined filters and displays non-interactive tabular results only."*

However, an unfinished/live candle should never be treated as a confirmed signal." ‚Üí Rewrite `confirm_if_needed` (`utils.py:414-430`) to evaluate the signal candle itself, **and** add an explicit platform-wide guard excluding the in-progress candle from indicator calculations.
